# Next.js Portfolio Template

A configurable personal portfolio template built with Next.js (App Router), Tailwind CSS, and Motion.

[![Deploy on Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/your-template-repo)

Live Demo: https://your-template-demo.vercel.app

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Project Structure

- `config/site.ts`: primary profile/site configuration
- `config/types.ts`: shared config/data types
- `data/experience.ts`: experience entries (JSON-style arrays)
- `data/projects.ts`: project entries (JSON-style arrays)
- `data/blog.ts`: blog page + post data
- `components/SeoMeta.ts`: reusable SEO metadata builder
- `app/globals.css`: theme tokens (light + dark variables)

## Edit Content

### Basic profile and section copy

Update [`config/site.ts`](./config/site.ts):

- `name`, `initials`, `role`, `description`, `email`
- `hero` text and CTA labels/links
- `about` paragraphs and `skillsIntro`
- `contact` eyebrow/title/button/email subject/body

### Experience entries

Update [`data/experience.ts`](./data/experience.ts):

- `experienceData` for the main portfolio

Each item is:

```ts
{
  company: string;
  role: string;
  period: string;
  description: string[];
}
```

### Project entries

Update [`data/projects.ts`](./data/projects.ts):

- `projectsData` for the main portfolio

Each item is:

```ts
{
  title: string;
  description: string;
  tech: string[];
  github: string;
  external: string;
  image: string;
}
```

## Edit Theme

Theme variables are in [`app/globals.css`](./app/globals.css):

- `:root` for light mode
- `.dark` for dark mode

Primary tokens:

- `--background`
- `--foreground`
- `--primary`
- `--primary-hover`
- `--primary-bg`
- `--muted`
- `--muted-foreground`
- `--card`
- `--border`

Dark/Light mode toggle is built in via [`components/ThemeToggle.tsx`](./components/ThemeToggle.tsx).

## Add Skills / Projects / Experience

### Add a skill

1. Open [`config/site.ts`](./config/site.ts).
2. Add a new entry to `about.skills`.

### Add an experience item

1. Open [`data/experience.ts`](./data/experience.ts).
2. Add a new object to `experienceData`.

### Add a project

1. Open [`data/projects.ts`](./data/projects.ts).
2. Add a new object to `projectsData`.

## SEO Metadata

SEO is centralized in `siteConfig.seo` inside [`config/site.ts`](./config/site.ts):

- `siteUrl`
- `title`
- `description`
- `keywords`
- `ogImage`

Metadata is generated by [`components/SeoMeta.ts`](./components/SeoMeta.ts) and consumed in:

- [`app/layout.tsx`](./app/layout.tsx)
- [`app/blog/page.tsx`](./app/blog/page.tsx)

## Deploy to Vercel

1. Push this project to GitHub.
2. Go to Vercel and import the repository.
3. Framework preset: `Next.js` (auto-detected).
4. Build command: `npm run build`.
5. Output is handled automatically by Next.js static export config.

## Connect a Custom Domain

1. Open your Vercel project.
2. Go to `Settings -> Domains`.
3. Add your domain (example: `portfolio.yourdomain.com`).
4. Copy DNS records shown by Vercel and add them in your domain registrar.
5. Wait for DNS verification, then set as primary domain.
